Dem config files
================

Get started
------------

* Clone the repo
* Make the `get_started` script executable  
`chmod +x get_started.sh`
* Set up config files for available tools  
`./get_started.sh`

Update files in repo
--------------------

* Make `update_files` script executable  
`chmod +x update_files.sh`
* Update all files  
`./update_files.sh
